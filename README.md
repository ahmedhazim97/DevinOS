# DevinOS

> The open-source engineering operating system for Devin and AI coding agents.

DevinOS provides reusable:

- Engineering Skills
- Rules
- Workflows
- Playbooks
- Templates
- Memory System
- Prompt Library

Its goal is to make AI software engineers more reliable, maintainable, and continuously improving across every repository.

## Vision

Build the largest high-quality reusable engineering brain for Devin.
